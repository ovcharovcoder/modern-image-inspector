# Changelog

All notable changes to the Modern Image Inspector VS Code extension will be documented in this file.
<br>

## [1.0.1] - 2025-12-25
### Changed
- Improved extension description and documentation.
- Updated README for better clarity and accuracy.

## [1.0.0] - 2025-12-24
### Added
- Initial release of Modern Image Inspector.
- Image inspection for PNG and JPG files.
- Display of image format, dimensions, and original file size.
- Estimated compression sizes for WebP (q75) and AVIF (q50).
- Ability to convert images to WebP and AVIF directly from the panel.
- Local and secure image processing using Sharp.
